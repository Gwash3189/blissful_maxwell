### Getting Started

1. Run `cp ./env.example ./env` and fill in the database connection string to point to a local postgres instance
2. Run the following code

```bash
npm run bootstrap && npm run start
```

Then open a browser to `localhost:3000`

### Exploring the data

```bash
npx prisma studio
```
